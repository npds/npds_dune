Paramètres système

L'utilisation de ce module nécessite une clef d'api que vous devez vous procurer ici. (Et cette obtention de clef nécessite un compte goggle.)
https://developers.google.com/maps/documentation/geocoding/get-api-key?hl=fr (cette clef permettra l'affichage de 25000 carte par 24 heures)
Le module est livré avec une clef générique que vous devez donc remplacer !..

Champ de table pour latitude et longitude

Présentent les champs disponibles dans la base de données pour le stockage des latitudes et longitudes des membres du site.

Géolocalisation des IP

Cette option permet d'activer ou de désactiver : - le stockage dans la base de donnée des IP des visiteurs,leur géolocalisation le nombre de visite et leur position de visite sur le site - l'affichage de ces données dans l'interface cette option peut ralentir un peu le site et stocker au fil du temps beaucoup de donnée dans la base. Vous pouvez à tout moment vider la base.

Interface carte

Type de carte

Vous pouvez définir ici le type d'affichage de la carte par défaut, parmi les 4 principaux types fournis par goggle

  • ROADMAP : carte routière
  • SATELLITE : vue satellite
  • HYBRID : vue satellite + carte routière
  • TERRAIN : relief
Unité des coordonnées

Permet de définir l'affichage par défaut des coordonées

  • dms : degrés minutes secondes (ex : 52°28' 58" S | 9°29' 32" E)
  • dd : degrés décimaux (ex : 63.391522 | 146.953125)
