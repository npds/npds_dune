Campo de tabla para latitud y longitud

Introduzca los campos disponibles en la base de datos para almacenar las latitudes y longitudes de los miembros del sitio.

Geolocalización de IP

Esta opción le permite habilitar o deshabilitar:
- almacenamiento en la base de datos de las IP de los visitantes anónimos, su geolocalización, el número de visitas y su posición de visita en el sitio
- visualización de estos datos en la interfaz
NB: esta opción puede ralentizar considerablemente la visualización del mapa (si ha iniciado sesión como administrador y su base de datos contiene muchas líneas (> 10.000 IP)) y almacenar muchos datos en la base de datos con el tiempo. Puede vaciar la base de datos en cualquier momento.

Cómo funciona:
La georreferenciación de IP es proporcionada por diferentes proveedores y complementa la base de datos del sitio.
Utilizamos 4 proveedores de geolocalización IP:
https://ipapi.co ==> máximo 1000 solicitudes por día, 30 000 por mes
https://api.ipdata.co => requiere clave API ==> máximo 1500 solicitudes por día
https://extreme-ip-lookup.com ==> máximo 20 solicitudes por minuto, 50 000 por mes
http://ip-api.com ==> máximo 150 solicitudes por minuto (prohibición)
Todos estos proveedores tienen un límite de uso gratuito o su número para poder garantizar un funcionamiento continuo siendo gratuitos...

Clave API de mapas de Bing

Esta clave le permite utilizar datos de mapas de Bing. Si desea utilizar estos datos, vaya a https://www.bingmapsportal.com y siga el proceso (crear una cuenta y luego obtener una clave)... y complete el campo.

Clave API de Mapbox

Esta clave (opcional) le permite utilizar datos de Mapbox. Si desea utilizar estos datos, vaya a https://www.mapbox.com y siga el proceso (crear una cuenta y luego obtener una clave)... y complete el campo.

Unidad de coordenadas

Le permite definir la visualización predeterminada de coordenadas

  • dms: grados minutos segundos (ej: 52°28' 58" S | 9°29' 32" E)
  • dd: grados decimales (ej: 63.391522 | 146.953125)
