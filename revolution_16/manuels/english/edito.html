EDITO


This module allows you to edit in real time the EDITO part of the site.

Using the HTML editor, you can copy and paste from Word (for instance).

The new Edito module (NPDS 4.8 P3) has brand new features such as :
 - Anonymous Edito and/or Member Edito,
 - Number of days it will be displayed (starting from date/hour of creation),
 - Displays Day time-text (08 AM=>08 PM) and Night time-text (08.01 PM =>07.59 AM),

Factory settings :
 - If there is no Edito for Members , members and anonymous will see the Anonymous Edito,
 - Daytime-text is only displayed from 08 AM to 08 PM if [day] is only selected,
 - Daytime-text will be displayed day and night if both boxes are selected ([day] and [day]) and if there's no Nighttime-text,
 - Nighttime-text is only displayed at night,
 - If Edito boxes are empty, no Edito will be shown.
