Editos

Editorial is the first thing visitors see when they arrive on your site, unless you prefer to change the link to the index page in preferences.

You can customize it, publish a different editorial day and night as well as choose to display it or not.

How it work's

The editorial has advanced features :

  • Anonymous Edito and/or Member Edito
  • Number of days it will be displayed (starting from date/hour of creation)
  • Displays Day time-text (08 AM=>08 PM)
  • Displays Night time-text (08.01 PM =>07.59 AM)
Note that these times can be changed in the preferences.

Factory settings :

  • If there is no Edito for Members , members and anonymous will see the Anonymous Edito
  • Daytime-text is only displayed from 08 AM to 08 PM if [day] is only selected
  • Daytime-text will be displayed day and night if both boxes are selected ([day] and [day]) and if there's no Nighttime-text
  • Nighttime-text is only displayed at night
  • If Edito boxes are empty, no Edito will be shown.

Note: the editorial can be displayed; wherever you want with the metaMot : !edito!
