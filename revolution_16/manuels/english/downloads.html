Download



This feature allows you to easily administrate the Download section. As many people use this page, you'd better read this just so you don't make any mistake.

Files available to download is displayed with all its informations (number, counter, URL, filename, ...) as well as the links that let you edit or delete the file.

Edit Download :
Click on Edit to display and edit the informations then click Save.
You can set the date and privileges in this pannel.
Don't forget to check your URL!

Delete a file :
Click on delete and validate.

Add a file to download :
- Enter the URL (external/internal to your site) or click on Browse.
. Browse allows you to 'browse' your site using a window and the Copy command updates the download form.
- The counter can be initialized with a value (don't brag...).
- If you want NPDS to calculate the correct size of the file, don't enter any value in the "file size" field

- Version, size, owners URL, owner's name are not mandatory.
- Catégory, description and privileges are mandatory.
- Nota bene :
. To add a category, just enter a new term in the field.
. The privilege option allows you to activate the download for users only or for anonymous.
