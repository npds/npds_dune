Users

This feature allows you to add users (members), to edit or delete them as well.

Add new user

Fill out the form. Required fields are marked with an * (Nickname, Email, Password).
Other fields are not mandatory. The hidden email adress is the one displayed to the other users (profile, forums, articles). NEVER PUT YOUR REAL EMAIL ADRESS in this field.
Click on Add user and NPDS will send an email with the password to the newly created user.

Member' role

You can attribute a specific role (for exemple : administrator, hot-line ...) to a member. the forum show this role and the administration of the role'list is in forum' config.

Group

A user can be a member of 0 to n groups and the group value is between 2(min) and 126(max). Groups are separated by a comma in the database.
- ex : 2,25 (member of groups 2 and 25)
The list of available groups is displayed and allows a multi-selection (shift / Ctrl).
The first choice allows to delete the membership to a group.
A group is a level-link that finds its meaning when creating blocks (see blocks).

Allow connexion

By default if the proper option is set to Yes in preferences, a member can log in as soon as he gets his password (email).
You can set it to off to forbid a member to access the website's members only features

Edit user

Enter his nickname, select Modify user and click on Ok.
You can therefore change everything (including the password).

Delete user

Enter his nickname, select Delete user and click on Ok.

Unsubscribe user

Enter his nickname, select Unsubscribe user and click on Ok.

Directory extraction

By clicking on Ok, you obtain a zip of the file containing information on all users. The zip contains a file in csv format.
