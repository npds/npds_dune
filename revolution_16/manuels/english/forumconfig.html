Forums configuration


Allow HTML :
Allows or forbid the use of HTML tags.

Allow Smilies :
Same with Smilies

Allow Signatures :
Same for signatures.

Treshold for 'hot' topics :
Allow to simply show with different gif, a hot topic.

Number of messages/page :
This is the number of posts/topic displayed in a page.

Topics per forum :
Number of topics in a forum (for one forum)

Activate the sort of the 'solved' contributions :
Activate the sort and the showing of the link : [ (Without or Only) Solved ] contributions

Maximum number of post per IP and per 30' period (0=system off) :
This option let you specified the maximum number of posts that a IP or Member can post :
- first : past 30 minutes
- second : past 90 minutes (in this case the value is the double)
0 or empty : system off

Allow upload :
To activate files upload in forums. Next, you have to allow this for each forum (Administration / Edit Forums).

Text for role X :
The forum is able to show a specific text for members with specific role (for exemple : administrator, hot-line ...). NPDS' multi-language tags are supported.
