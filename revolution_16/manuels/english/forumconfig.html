Forums configuration

Allow HTML:

Allows or prohibits (removes) the interpretation of HTML tags.

Allow Smilies:

Authorizes the insertion of Smilies in the Forums.

Authorize Signatures:

Allows you to authorize the use of signatures for members in forum publications.

Threshold for 'hot' topics:

Allows you to define the threshold above which a subject is considered "hot".

Number of contributions per page:

This is the number of contributions per subject that will be presented per page for a single subject.

Topics by forum:

This is the number of topics per forum that will be activated for a single forum.

Enable sorting of 'resolved' contributions:

Activates sorting and display of the link [(Only / Without) Resolved].

Maximum number of contributions per IP per 30 minute period (0=system inactive):

This option allows you to specify the maximum number of posts that the same IP address OR the same member can post during:

  • a first period of 30 minutes
  • a second period of 90 minutes (in this case double this value is used)
  • 0 or empty means disabling the system

Enable uploading in the forums:

Enables the ability to store files directly in the forums. Then you can authorize this function only in certain of your forums (see Administration / Editing of Forums).

Enable hidden texts in forums:

Allows you to activate the possibility of hiding part of the text of a post in the forums using the [hide] My text to hide [/hide] tags.

Text for role X:

The forum is capable of displaying specific text based on a member's role. Multi-language tags are supported in these areas.
