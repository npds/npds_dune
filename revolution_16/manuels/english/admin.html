Admin

Introduction

NPDS is an Open-Source French portal generator under the free GNU-GPL license using PHP and MySql technologies which has a multi-language feature that can be configured in the preferences. By default, the portal is in French.

The content and interface of this page obviously depends on the rights you have as an administrator (the super administrator will therefore have access to all interfaced functions). The general interface is divided into 3 main blocks

Alerts and notifications

The first block brings together alerts/notifications allowing you to process pending administrator tasks. (proposed articles, scheduled articles, users awaiting validation, new links, broken links, etc.)

Functions

To manage NPDS, you have access to numerous buttons which are grouped by row according to a classification by type. They appear in the form of icons or text depending on the graphical interface settings defined in the preferences.

  • The buttons in the first row allow you to manage the content of your site, once you have clicked on a button you must click on the online manual button to obtain the documentation in relation to the open function.
  • The second row allows you to manage the portal's users
  • The third row allows you to manage everything related to communication
  • The fourth row allows you to manage the interface of the blocks
  • The fifth row allows you to act on the system, the preferences link is one of the most important links. Note the link which allows you to install a module and to know which modules are already installed or not.
  • Finally the last row which allows you to manage modules with administration.

Last x articles

  • Displays the list of the last x articles published on your site
  • The first record is the last published article
  • Certain articles can be indicated in archive, these are the articles having passed their end of publication date.
  • To put these articles back in the feed, simply edit them, click on the 'change date' option and modify their publication end date.
  • Click on the article to view it or use the Edit, Delete links.
  • If you want to Edit/Delete an old article, enter its ID and select the appropriate action.
