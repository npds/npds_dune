IP

This core module allows you to view IPs banned or in the process of being banned by npds antispam. It also allows you to manually ban IPs (v4 or v6) or IP ranges (v4).

Each line must only contain an IP address (v4 or v6) of the form:
a.b.c.d|X (ex. v4: 168.192.1.1|5);
a:b:c: d:e:f:g:h|X (ex. v6: 2001:0db8:0000:85a3:0000:0000:ac1f:8001|5).
If X >= 5 then this IP address is banned and access will be denied!

You can also ban IP address ranges (v4) in this form:
x.x.%|5 or x.x.x.%|5
