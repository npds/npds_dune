Informations Channels (Headlines)


You can activate informations channels from other sites.
These channels are not text files but RDF/XML files.
Warning : using this feature may slow down your server.
Every channels are refreshed every hour. Don't change this value as it may use up most of your bandwidth

Add a channel :
Fill the fields (Title, URL, URL RDF/XML, status). If you activate a channel, a new box will appear. Inactive boxes won't appear.

Change a channel status :
Just click on Edit and select the status on the next page.

To delete a channel, click on... delete.

If your site is behind a Proxy Server configure the proxy.conf.php file (root of your site) to allow the access to one or more informations channels.

Configuration of boxes that host information channels :
in the Manage blocks (left/right), you can add:
- function#headlines
=> in this case, all active channels are displayed one after another.
OR
- function#headlines
- params#$ID_of_channel
=> in this case only the ID channel "ID_of_channel" is displayed. You can set as many blocks as you want...
