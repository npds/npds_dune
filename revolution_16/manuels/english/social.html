Social networks

Here you can determine which social networks will be available to your users.
By default you have facebook, X (twitter), snapchat, linkedin, github.
You can therefore Add, Edit, Delete a network.
To add a network three elements are necessary:

  • the name
    - example: tiktok
  • url
    - note: with a / ending the url
    - example: https://www.tiktok.com/
  • an icon
    - note: must correspond to the name of the web font class of fontawesome.
    - example: tiktok
