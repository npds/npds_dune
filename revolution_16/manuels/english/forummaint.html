Forums Maintenance


Set all Topics as read :
You can set them all as read. This option can re-initialize the forum_read MySql table.

Massive suppresion Topics :
Allows to delete the content of forums (with a eventually a date). The interface displays, by forum,Topic's IDs.

Two modes :
 - Select as much topics as necessary and click on "delete Topics" (Warning! You can't go back!).
 - Select a Topic, its title and the last Post.
Then you can delete it.

Synchronize Forums :
Allows the Re-Synchronization of the forum_read table (that deals with read marks). Sometimes the synchronization is lost.

This may take a while (depending on the number of forums and topics)

Merge Forums :
simply merge all the posts of one forum to another.
This may take a while (depending on the number of forums)
