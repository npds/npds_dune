Forums maintenance

Mark all Topics as read:

Allows you to position all Posts in all Topics in all Forums as read.
This function also allows you to reset the forum_read table.

Massively delete Topics:

Allows the mass deletion of Forum content (entering a date YYYY-MM-DD allows you to make a selection of the oldest posts for example). The interface presents, per forum, the topic ID.

Two modes:

  • Select as many Topics (check boxes) as necessary and click on the “Massively Delete Topics” button
    Please note, no confirmation requested.
  • Select a Topic, its title as well as the last attached Post is displayed.
    You can then delete it.

Synchronize Forums:

Allows re-synchronization of the forum_read table (which manages reading markers). It may happen that this table is no longer in sync with the topics of a forum.
This operation can be relatively long (depending on the size of your forums).

Merge forums:

Simply allows the transfer of all posts from one forum to another.
This operation can be relatively long (depending on the size of your forums).
