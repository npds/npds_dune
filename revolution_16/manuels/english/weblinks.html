Links

The web link database is an important feature of NPDS. Its maintenance is simple but requires a certain rigor.
It allows the creation of a Yahoo type index with a search engine independent of the rest of the site.

Links awaiting validation

You will only have this option if a link is pending (sic ...).
You can modify the title, the link and its comment, the name and email of the depositary, the category.
You can finally choose to publish the link by clicking on the Add button or delete it.

Clean the link database:

This option allows you to reset the votes for all links in the database.

Broken link submission:

The options applicable to the list are:

  • Ignore clears all requests for a given link.
  • Clear allows you to clear broken links and reviews for a given link.

Link modification submission:

For each link, you will have the origin (currently stored in the database) and the proposed modification.

Applicable options are:

  • Ignore clears all requests for a given link.
  • Clear allows you to clear broken links and reviews for a given link.
  • click on the Author or Owner link to send an Email.

Add a main category:

To structure the index, enter a title and a description (sorted in alphabetical order).

Add a subcategory:

For index structuring, enter a subcategory assigned to a main category (single level / alphabetical ordering).

Add a new link:

Enter the title (followed by html tags (br, img src (local or remote), ...) if you wish), the URL, the category (and subcategory), description, owner name and email. Click the Add button.

Edit category:

This function allows you to modify the name of a category and/or subcategory.
It also allows you to delete a category and/or subcategory and all attached elements (with confirmation message).

Edit links:

NPDS presents the list of links currently available in the database. Click a link to edit it.
You can modify all information, including the number of hits (to position a link more advantageously...).
You can also delete the selected link.
