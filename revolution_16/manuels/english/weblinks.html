Web links


This base is an important feature of NPDS. It only requires a 'daily' efficient maintenance.
It allows the creation of a Yahoo-like index with its own search engine.

Waiting links :
When there's a link waiting for your approval (lol ...).
You can change its title, the URL, the commentary, the name and email of the one who submitted the link, the category.
Then you can add the link (Add) or not (Delete).

Clean Links DB :
Set to zero the votes database (votes about links, it has nothing to do with the poll and surveys).

Broken links submission :
These are the options :
- Ignore allows to delete all demands for a given link.
- Delete allows to delete broken links and comments for a given link.

Links change Submission :
For each link, you have the original link (the one stored in the database) and the new URL.
Options are :
- Ignore allows to delete all demands for a given link.
- Delete allows to delete broken links and comments for a given link.

Add main category :
In order to have a good structured index, enter a title and a description (alphabetically sorted).

Add sub-category :
In order to have a good structured index, enter a sub-category related to a main category (one level / alphabetically sorted).

Add new link :
Enter the title (you can add html code (br, img src (local or no), ...) if you want to ), URL, category (and sub-category), description, name of the one who submitted the link and his email. Click on Add.

Modify category :
You can change the name of a category or sub-category.
You can also delete a category or sub-category with all the following elements (confirmation message).

Modify links :
NPDS shows you all the links it has stored in its database. click on a link to edit it.
You can change everyhting from description to the number of hits (if you need to 'help' a friendly link...). You can also delete the selected link.
