Reviews

This function allows the administration of the “reviews” application of your site.

You can modify the title of the reviews page as well as the description (message) relating to it. To finish, click on the Save Changes button.

Reviews awaiting validation:

Displays the list of latest reviews posted by users and allows you to publish them or not.

Suggest a new review:

Displays the same page as is available to users.

You can modify or delete directly in the Reviews function if you are logged in as an administrator.
