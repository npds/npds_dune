Faqs

Setting up FAQS allows you to gain time! Some people may post messages in the forums again and again (i.e. : how to install your product? Why isn't the X function working under Windows, ...).

The NPDS Faqs features requires at least one category.

From the selected category, you can edit (create, edit and delete) every Q&A
The answer field allows the use of HTML.
Faqs can display Php code, Perl, C....
You need to put your "source" between '[]' like this : [code]your_source[/code].
This function comes from phpBB.
