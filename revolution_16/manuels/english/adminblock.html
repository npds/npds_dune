Admin block

If you are an administrator, you have a specific block (arranged according to the theme).

This block will automatically display in the form of an icon various notifications of actions to be carried out by the superadministrator or by the administrators having rights to the functions issuing these notifications. (eg: user, review, link awaiting validation... etc...).
It will also display notifications (of two types: note and security.) coming directly from npds.org and an alert if the version used is not the latest published.

You cannot delete this block and you can only have one.

Be creative and insert your useful links in this block such as: New Article, Banners etc...

You can also insert HTML code (but not PHP...).
