Topics Manager

You need to have set up a few topics to create articles. The topics manager allows you to manage your topics. This is a great feature of NPDS

Every topic requires a title, a description, an image (not required) and one (or more) administrator(s).

Add a topic :
- Topic Name : this is for NPDS. Enter just a name without spaces - max: 20 characters.

- Topic Text : This will be the Displayed Text. You can use both lower and upper case and spaces.

- Image : You don't have to but if your topic requires an image enter the name of the file (i.e. : npds.gif). This file has to be in the images/topics folder of your site

- Administrator : NPDS allows to put one or more admin for a topic.
If you leave the field blank, ONLY SuperAdmins will be allowed to publish in this category.
If it has one or more (use , (commas) to separate the admins) => Only the admin(s)(and SuperAdmin) will get to publish articles in this topic.

Edit a Topic :
Click on the image or the text. The next page will let you change the Title,the name, the description and the image.
You can delete a topic but this will delete all nested articles too!

You can add links to a topic, like a link to www.microsoft.com for a Windows XP topic. This link will be available when viewing an article from that category.
