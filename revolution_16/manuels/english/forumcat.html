Edit Forums


You can edit and administrate your forums with this feature..
The first list displays the categories and active forums

Edit Forums :
This second page shows the forums from that category
You can add a forum :
- Forum Name
- Description (keep it simple)
- nickname of moderator(s) (required) : if you want to add more moderators to the same forum, use : nickname, nickname1,nickname7...

- Access level (Post) :
=> Anonymous post allowed
=> Registered users post only
=> Moderators post only
=> Closed ( forum is closed )

- Forum type : Public, Private, PHP Script, extended text, Groups or Hidden
=> Public = everyone can access the forum
=> Private = you'll need a password to enter the forum
=> PHP Script = This Forum allows to display code (php, perl, c, ...).
Warning : some functions do not work with this forum (Smilley, ...)
=> Extended Text = You can associate a SFORM to this forum
=> Groups = You can make this forum only for the members of N groups
=> Hidden = Only members can see it
=> This last field uses some values :
    -> Password (if Private Forum )
    -> Form filename(if Extended Text Forum)
       in modules/sform/forum (i.e. : form.php)
    -> Members groups ID (if Groups Forum) (ID : 5,12,56 for instance)

- Forum Mode : Standard or Tree
=> The tree mode allows a particular display but requires more power from the server

- Attachment :
=> if you have allowed the forum upload feature, you can specifiy which forum can use this function.

Make a redirection to an external forum (a link to call it out loud), just enter the URL in the field for the Forum Name!

Sub-Forum :
A forum isn't visible if it's name start with <!>. One new meta-mot give you the reading status of one forum : forum_subfolder(ID_of_forum).
=> Imagine one forum with id = 3 and with this name : "<!>Only for NPDS" / Imagine the comment-text of forum with id 1 like : forum_subfolder(3) <a href="viewforum.php?forum=3">Only for NPDS</a>


You can edit a forum to make changes and you can also delete it.

Add category :
You can add a category in which you can add forums. Click on Add to validate.

Edit (category) :
You can edit the category (the title AND ID (to sort the categories). Click on Ok to validate.

Delete (category) :
You can delete a category with all the forums and messages related to it.
