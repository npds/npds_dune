Logs

The Session List tab allows you to view the active sessions on your site in detail.

The Logs List tab allows you to view and analyze the logs issued by the portal. They are divided into two categories, so-called security logs and logs tracking downloads. In the table the Info function allows you to obtain detailed information on the IP.

The tab contains commands for:

  • send the security.log file by email to the superadmin.
  • empty the security.log file
  • empty temporary upload files (modules/upload/tmp/)
