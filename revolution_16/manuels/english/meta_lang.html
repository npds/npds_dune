META-LANG

META_LANG what is it for?

META-LANG is a language specific to NPDS. It is composed of words (the META-MOTs) allowing, in complete security, to call to the location where you have entered this META-MOT (in a news item, an article, a post, a comment or the editorial of your site or even a static page), the result of internal NPDS functions.

There are 5 types of META-MOT in use for your site

Types of META-MOT

  • docu
    Will be used by the CMS for very specific processing and/or online documentation.
    Currently only ^ is present in this category
    You will not be able to create this type of meta-word
  • meta
    In use in almost all NPDS, will be used to call the result of a specific function which will be displayed at the location of its entry.
    This is the most important category of NPDS and allows the implementation of very advanced functions.
    There are two forms of writing (in the script area) depending on the level of complexity of what you want to achieve:
    • $cmd=... single-line PHP instruction...;
    • function MM_your-meta-word () {
      php code
      php code
      php code
      return (your-return);
      }
  • mot
    Will be used to display a word (association of letters or numbers) that can perform a basic HTML-type function such as calling a link, spelling correction, management of abbreviations...
    This category is very useful for increasing productivity.
  • smil
    Will be used to integrate smilies and/or shortcuts to images.
  • them
    Will be used to document internal NPDS functions related to themes.
