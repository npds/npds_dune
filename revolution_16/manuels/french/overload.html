Ce lien vous permet de modifier la configuration d'une fonction importante de NPDS

le SuperCache

SuperCache est un système de cache Php/Html vous permettant d'optimiser fortement les performances de votre site.

Vous pouvez :

  • Activer ou Non le Supercache
  • Définir pour certaines fonctions, la durée du cache (rétention en secondes)
  • Vider le contenu du répertoire cache lors, par exemple, d'une mise à jour importante
