Administration - Modules

Permet d'installer et de désinstaller les modules. L'interface présente un tableau simple avec l'icône, le nom du module et les fonctions.

Les fonctions

Pour les modules avec installation-automatique
  • : installer le module.
  • : désinstaller le module.
    Attention cette commande détruit ce qui a été créé par l'installation automatique du module :
    • les éventuelles tables de la base ainsi que les données s'y trouvant.
    • les éventuelles fonctions d'administration et les droits attachés
    • les éventuels blocs relatifs au module.
    • les éventuels meta-mots
  • Ces deux commandes sont disponibles alternativement que si le module dispose d'une installation automatique définie dans le fichier /modules/mon_module/install.conf.php. Elles conduisent donc à l'interface détaillée et progressive d'installation ou de désinstallation de votre module.
Pour les modules sans installation-automatique
  • : marquer le module comme installé.
  • : marquer le module comme désinstallé.
  • Ces deux commandes sont disponibles alternativement que si le module ne dispose pas d'une installation automatique. Elles ne servent donc qu'à modifier l'état du module installé/désinstallé dans la base de données et sa notification dans l'interface. Vous devez donc installer/désinstaller le module en vous référant au fichier install.txt de l'archive du module.
