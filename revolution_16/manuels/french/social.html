Réseaux sociaux

Ici vous pouvez déterminer quels réseaux sociaux seront disponibles pour vos utilisateurs.
Par défaut vous disposez de facebook, X (twitter), snapchat, linkedin, github.
Vous pouvez donc Ajouter, Editer, Supprimer un réseau.
Pour ajouter un réseau trois éléments sont nécessaires :

  • le nom
    - exemple : tiktok
  • l'url
    - note : avec un / terminant l'url
    - exemple : https://www.tiktok.com/
  • une icone
    - note : doit correspondre au nom de la classe du web font de fontawesome.
    - exemple : tiktok
