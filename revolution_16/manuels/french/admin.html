1. Menu

Introduction

NPDS est un générateur de portail Français en Open-Source sous licence gratuite GNU-GPL utilisant les technologies PHP et MySql qui possède un dispositif multi-langues paramétrable dans les préférences. Par défaut, le portail est en français.

Les fonctions

Pour gérer NPDS, vous avez accès à de nombreux boutons qui sont rassemblés par rangée.

  • La première rangée est constituée de divers boutons correspondant à des dispositifs d'alerte de l'administrateur. Les articles proposés et les articles programmés sont accessibles ici.
  • Les boutons de la deuxième rangée vous permettent de gérer le contenu de votre site, une fois que vous avez cliqué sur un bouton il vous faut cliquer sur le bouton manuel en ligne pour obtenir la documentation en rapport avec la fonction ouverte.
  • La troisième rangée vous permet de gérer les utilisateurs du portail
  • La quatrième rangée vous permet de gérer tout ce qui concerne la communication
  • La cinquième rangée vous permet de gérer l'interface des blocs
  • La sixième rangée vous permet d'agir sur le système, le lien préférences est un des liens les plus importants. A noter le lien qui vous permet d'installer un module et de savoir les modules qui sont déjà installés ou non.
  • Enfin la dernière rangée qui vous permet de gérer les modules dotés d'une administration.

Les x derniers articles

  • Affiche la liste des x derniers articles publiés sur votre site
  • Le premier enregistrement est le dernier article publié
  • Certains articles peuvent être indiqués en archive, ce sont les articles ayant dépassés leur date de fin de publication.
  • Pour remettre ces articles dans le flux, il suffit de les éditer, de cliquer sur l'option 'changer la date' et de modifier leur date de fin de publication.
  • Cliquez sur l'article pour l'éditer ou utilisez les liens Editer, Effacer.
  • Si vous voulez Editer/Effacer un ancien article, saisissez son ID et sélectionnez l'action appropriée.
