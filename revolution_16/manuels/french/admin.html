Administration - Accueil

Introduction

NPDS est un générateur de portail Français en Open-Source sous licence gratuite GNU-GPL utilisant les technologies PHP et MySql qui possède un dispositif multi-langues paramétrable dans les préférences. Par défaut, le portail est en français.

Le contenu, l'interface de cette page dépend bien entendu des droits que vous disposez en tant qu'administrateur (le super administrateur aura donc accès à toutes les fonctions interfacées). L'interface générale est divisée en 3 blocs principaux

Les alertes et notifications

Le premier bloc rassemble les alertes/notifications vous permettant de traiter les tâches administrateur en attente. (articles proposés, articles programmés, utilisateurs en attente de validation, nouveau liens, liens rompus etc ...)

Les fonctions

Pour gérer NPDS, vous avez accès à de nombreux boutons qui sont rassemblés par rangée en fonction d'une classification par type. Ils se présentent sous forme d'icônes ou de texte en fonctions du paramétrage de l'interface graphique défini dans les préférences.

  • Les boutons de la première rangée vous permettent de gérer le contenu de votre site, une fois que vous avez cliqué sur un bouton il vous faut cliquer sur le bouton manuel en ligne pour obtenir la documentation en rapport avec la fonction ouverte.
  • La deuxième rangée vous permet de gérer les utilisateurs du portail
  • La troisième rangée vous permet de gérer tout ce qui concerne la communication
  • La quatrième rangée vous permet de gérer l'interface des blocs
  • La cinquième rangée vous permet d'agir sur le système, le lien préférences est un des liens les plus importants. A noter le lien qui vous permet d'installer un module et de savoir les modules qui sont déjà installés ou non.
  • Enfin la dernière rangée qui vous permet de gérer les modules dotés d'une administration.

Les x derniers articles

  • Affiche la liste des x derniers articles publiés sur votre site
  • Le premier enregistrement est le dernier article publié
  • Certains articles peuvent être indiqués en archive, ce sont les articles ayant dépassés leur date de fin de publication.
  • Pour remettre ces articles dans le flux, il suffit de les éditer, de cliquer sur l'option 'changer la date' et de modifier leur date de fin de publication.
  • Cliquez sur l'article pour le voir ou utilisez les liens Editer, Effacer.
  • Si vous voulez Editer/Effacer un ancien article, saisissez son ID et sélectionnez l'action appropriée.
