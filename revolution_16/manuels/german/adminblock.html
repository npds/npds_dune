Block Administration

Als Administrator können Sie eine(n) Box/Block aktivieren (links oder rechts, je nach dem, welches Template Sie verwenden). Sie können diese(n) Box/Block nicht löschen, oder duplizieren.
Seien Sie kreativ und nutzen Sie es um zum Beispiel hilfreiche Links zu plazieren, wie: Neue Artikel, Banner, ...
Sie können auch HTML nutzen (aber kein PHP ...).
