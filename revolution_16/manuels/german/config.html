Export-News sind zwei Textdateien, die sich im Seitencache befinden. Standardeinstellung ist = Nein.

Diese Dateien können Inforamationen von anderen Seiten enthalten, unter Verwendung einfacher Strukturen. :
-ultramode.txt ist eine Textdatei in der Artikel getrennt werden durch %%
-net2zone.txt ist eine XML Text Datei für net2zone.

Start Seite

Diese Funktion erlaubt die Einstellung der Startseite (Home). Standard ist index.php Sie können aber auch eine andere Seite einstellen zum Beispiel :
- links.php
- faqs.php
- static.php?op=npds.txt&npds=1 ... (npds kann sein -1 : keine Boxes/Blöcke, 0 : left box/block, 1 : linke und rechte Boxen/Blöcke)

- EDITO :

  • index.php?op=edito : zeigt Edito UND die News an (ohne "Nächste Seite" Link)
  • index.php?op=edito-nonews : zeigt Edito OHNE die News an (mit "Nächste Seite" Link)
  • index.php?op=newindex : zeigt News OHNE Edito (mit "Nächste Seite" Link)
  • index.php?op=edito-newindex : zeigt Edito UND die News an (mit "Nächste Seite" Link)
  • index.php?op=newtopic&topic=X : zeigt News im Zusammenhang mit Thema'id X OHNE Edito (mit "Nächste Seite" Link)
