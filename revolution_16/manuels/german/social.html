Sozialen Netzwerke

Hier können Sie festlegen, welche sozialen Netzwerke Ihren Benutzern zur Verfügung stehen.
Standardmäßig sind Facebook, X (Twitter), Snapchat, LinkedIn und Github verfügbar.
Sie können daher ein Netzwerk hinzufügen, bearbeiten, löschen.
Um ein Netzwerk hinzuzufügen, sind drei Elemente erforderlich:

  • der Name
    - Beispiel: tiktok
  • URL
    - Hinweis: Mit einem / endet die URL
    - Beispiel: https://www.tiktok.com/
  • ein Symbol
    - Hinweis: Muss dem Namen der Web-Font-Klasse von Fontawesome entsprechen.
    - Beispiel: tiktok
