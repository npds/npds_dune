LNL - Kleiner NewsLetter


Dieses Modul erlaubt Ihnen die Administration eines kleinen Newsletter für Ihre Seite.

Seine Eigenschaften sind :
 - Administration des LNL in 3 Abschnitten : Header, Body, Footer,
 - Administration des LNL als Text, oder als HTML,
 - LNL für reg. Benutzer oder/und Besucher die Ihre E-mail Adresse angeben zur Nutzung des LNL-Blocks
 - LNL'Status.

Grundsätzlich arbeitet der LNL so :
 - Erstellen des Header (Text oder HTML),
 - Erstellen des Body (Text oder HTML),
 - Erstellen des Footer (Text oder HTML),
 - LNL testen (geben Sie eine Nummer für jedes Element ein),
   Mit dem Test erhalten sie eine Vorschau des LNL
 - LNL senden (Geben Sie einen Betreff und die Benutzer ein, an die des LNL gesendet werden soll.)

Sie können Teile hinzufügen, ändern, oder löschen.

Sie können den Status des LNL ansehen: bereits gesendet, oder die folgenden Arten :  
- All => LNL wurde an alle Benutzer gesendet, die angemeldet sind,
- Mbr => nur an reg Mitglieder,
- Out => nur an Benutzer, die sich für den LNL Block angemeldet haben

Status der LNL Block Benutzer (OK => Senden Sie mir den LNL).

Die E-mail Funktion muss bei Ihrem Hoster aktiviert sein, damit Sie diese Funktion nutzen können..
Wenn es mehr als 50 Abonenten gibt, wird der LNL in Gruppen zu 50 verschickt, um die Serverlast zu verringern.
