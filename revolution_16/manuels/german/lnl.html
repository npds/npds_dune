LNL - Kleiner NewsLetter

Mit diesem Modul können Sie einen Newsletter für Ihre Website verwalten.

Die Hauptfunktionen dieses Moduls sind:

  • Verwaltung eines Newsletters in 3 Teilen: Kopfzeile, Hauptteil, Fußzeile
  • Verwaltung eines Newsletters im Text- oder HTML-Modus
  • Verwalten Sie Sendungen für Mitglieder, Interessenten oder beides
  • Überwachung gesendeter LNLs und der Liste potenzieller Kunden

Typischerweise ist die Betriebsart:

  • Erstellen einer Kopfzeile (Text oder HTML)
  • Erstellen eines Körpers (Text oder HTML)
  • Erstellen einer Fußzeile (Text oder HTML)
  • LNL-Test (geben Sie die Nummer für jedes Element ein)
    Mit diesem Test können Sie die Darstellung Ihres LNL
    validieren
  • Senden der LNL (Geben Sie den Betreff und die Empfänger ein).

Sie können die „Komponenten“ Ihres LNL hinzufügen, ändern oder löschen.

Sie erhalten einen Status der gesendeten LNLs mit einer bestimmten Menge an Informationen und insbesondere dem Typ:

  • Alle => LNL wird an alle Mitglieder und alle Interessenten gesendet
  • Mbr => nur für Mitglieder
  • Out => nur für Interessenten
sowie die Liste der Interessenten (Status OK => Der Benutzer möchte den Brief erhalten, NOK, er möchte nicht).

Natürlich muss die Mail- bzw. E-Mail-Funktion bei Ihrem Gastgeber aktiv sein, um diese Funktionalität nutzen zu können.

Über 50 Mitglieder wird die LNL alle 10 Sekunden in „Paketen“ von 50 gesendet, wodurch eine Überlastung Ihres Mailservers vermieden wird.
