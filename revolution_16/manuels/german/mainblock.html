Haupt Block


Jede Seite hat ein Hauptmenu! Sie können es nicht löschen und Sie können nur einen Block verwendenE.

Das ist das wichtigste Menu für Ihre Seite, weil es alle wichtigen Abschnitte der Seite enthält.
Sie können HTML verwenden, aber kein PHP.

Schlüsselfunktionen:

  • a href=static.php?op=premier.txt => Static page
  • a href=sdv.php?sdv=Annonces => Classifieds (?sdv=[name of the room])
  • a href=sdv.php?sdv=Name%20of%20the%20Room => Classifieds (%20 stands for space)
  • a href=index.php?op=index.php =>News (op=[Opening page])
  • a href=forum.php => Forums
  • a href=topics.php => Topics
  • a href=sections.php => Sections
  • a href=reviews.php => Reviews
  • a href=links.php => Links
  • a href=download.php => Download
  • a href=memberslist.php => Memberslist
  • a href=faq.php => FAQ
  • a href=friend.php => Tell a friend
