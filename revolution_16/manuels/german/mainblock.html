Hauptblock

Alle Websites haben ein Hauptmenü! Sie können diesen Block nicht löschen und nur einen haben.

Dieser Block ist von grundlegender Bedeutung, um auf die Funktionen Ihrer Website hinzuweisen.
Sie können in diesem Block HTML-Codes verwenden, jedoch kein PHP.

Erinnerung an die wichtigsten Links:

  • Statische Seite
  • Neuigkeiten (op=[Startseite])
  • Foren
  • Themen
  • Rubriken
  • Weblinks
  • Bewertungen
  • Downloads
  • Verzeichnis
  • Sitemap
