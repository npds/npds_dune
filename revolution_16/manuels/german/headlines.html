Informations Kanäle (Headlines)

Sie können Informations Kanäle von anderen Seiten aktivieren.
Diese Kanäle sind keine Textdateien, sondern RDF/XML Dateien.
Warnung : dieses Modul kann die Geschwindigkeit Ihrer Webseite negativ beeinflussen.
Alle Kanäle werden ständlich erneuert. Verändern Sie diese Einstellung nicht, denn es beansprucht schon so genug Recourcen.

Einen Kanal hinzufügen

Füllen Sie die Felder aus (Titel, URL, URL RDF/XML, Status).
Wenn Sie einen Kanal aktivieren, wird eine neue Box angezeigt. Inaktivie Kanäle werden nicht angezeigt.

Einen Kanal eistellen

Klicken Sie auf editieren und wählen Sie den Status der Seite aus.
Um einen Kanal zu löschen, klicken Sie auf... löschen.
Wenn Ihre Seite hinter einem Proxy Server liegt, konfigurieren Sie dieproxy.conf.php(root Ihrer Seite) um den Zugang zu einem oder mehreren Kanälen zu erlauben.

Konfiguration der Boxen, in denen die Kanäle angezeigt werden sollen

In der Block Administration (links/rechts), können Sie hinzufügen:

  • function#headlines
    => in diesem Fall werden alle Kanäle einer nach dem anderen angezeigt.
  • function#headlines
    params#$ID_of_channel
    => in diesem Fall wird nur die ID des Kanals angezeigt "ID_of_channel" . Sie können soviele Kanäle einfügen, wie Sie wollen.
