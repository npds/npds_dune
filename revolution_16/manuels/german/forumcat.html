Forum editieren

Die zweite Liste zeigt die Foren in den Kategorien
Sie können ein Forum hinzufügen :
- Name des Forums
- Beschreibng (einfach und klar)
- Nickname(n) des/der Moderators(en) (Pflicht) : Wollen Sie mehrere Moderatoren angeben, machen Sie es bitte so : Nickname, Nickname1,Nickname7...

- Zugangsberechtigungen (schreiben) :
=> Gastbeitr�ge erlauben
=> Nur für registrierte Benutzer
=> Nur für Moderatoren
=> Geschlossen ( Forum ist geschlossen )

- Forum Typ : öffentlich, Privat, PHP Script, erweiterter Text, Gruppen oder verborgen
=> Public = Jeder kann das Forum sehen
=> Private = Sie brauchen ein Passwort, um das Forum zu betreten
=> PHP Script = Das Forum erlaubt Scripte (php, perl, c, ...).
Warnung : einige Funktionen sind nicht verfügbar (Smilley, ...)
=> Erweiterter Text = Sie können ein SFORM diesem Forum zuordnen
=> Gruppen = Sie können das Forum nur Mitgliedern einer bestimmten Gruppe zug�nglich machen
=> Verborgen = Nur Mitgliedern ist der Zugang gestattet
=> Das letzte Feld erfordert einige Eingaben :
    -> Password (für privates Forum )
    -> Name des Formulars(für erweitertes Text Forum)
       in Modulen/Sform/Forum (z.B. : form.php)
    -> Mitglieder Gruppen ID (für Gruppen Forum) (ID : 5,12,56 zum Beispiel)

- Forum Modi : Standard oder Baum
=> Die Baumansicht ist detailierter, erfordert aber mehr Serverleistung

- Dateiupload :
=> Wenn Sie den Dateiupload erlauben, können Sie hier einstellen für welche Foren das gilt.

Verlinken Sie ein externes Forum geben Sie eine URL für den Forennamen ein!

Unterforum :
Ein Forum bleibt unsichtbar wenn es so beginnt <!>. Ein meta-Wort gibt Ihnen den Lesestatus für ein Forum : forum_subfolder(ID_of_forum).
=> Zeige ein Forum mit der ID = 3 und mit dem Namen : "<!>Only for NPDS" / Zeige den Kommentartext des Forums mit ID= 1 wie : forum_subfolder(3) <a href="viewforum.php?forum=3">Only for NPDS</a>


Sie können ein Forum editieren, aber auch löschen.

Kategorie hinzufügen

Sie können eine Kategorie hinzufügen worin Sie dann Foren hinzufügen können. Klicken Sie auf "ADD".

Edit (Kategorie)

Sie können eine Kategorie editieren (den Titel und die ID (um die Kategorien zu sortieren). Klicken Sie dann auf OK.

Löschen (kategorie)

Sie können eine Kategorie und alle darin enthaltenen Forum löschen
