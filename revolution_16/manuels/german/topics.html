Themen Manager

Sie müssen zunächst ein, oder mehrere Themen anlegen, dem (denen) Sie dann Ihre(n) Artikel zuordnen können. Der Themen Manger erlaubt Ihnen Ihre Themen zu verwalten.

Jedes Thema erfordert einen Titel, eine Beschreibung, ein Bild (nicht unbedingt erforderlich) und einen, oder mehrere Administrator(en).

Thema hinzufügen :
- Name des Themas : das ist für NPDS. Geben Sie einen Namen ohne Leerzeichen ein - maximal: 20 Buchstaben.

- Text des Themas : Das ist der angezeigte Text. Sie können Gross- und Kleinbuchstaben, sowie Leerstellen verwenden.

- Bild : Sie können, müssen aber nicht, ein Bild verwenden (z.B. : npds.gif). Dieses Bild muss unter images/topics gespeichert werden

- Administrator : NPDS benötigt mindestens einen Administrator für jedes Thema.
Wenn Sie ddas Feld leer lassen, kann nur der Superadmin die Themen bearbeiten.
Gibt es mehrere Administratoren (benutzen Sie , (Kommas) um die Admins zu trennen) => Nur der/die Admin(s) und der Superadmin können die Themen bearbeiten.

Ein Thema bearbeiten :
Klicken Sie auf das Bild, oder den Text. Auf der nächsten Seite können Sie den Titel, den Text und das Bild ändern.
Sie können auch ein Thema löschen. Das löscht dann aber auch alle damit verbundenen Artikel! Seinen Sie also vorsichtig!

Sie können einem Thema auch einen Link zuordnen. Beispielsweise www.microsoft.com Für ein Windows 7 Thema. Der Link wird angezeigt, wenn ein Artikel des Themas angeklickt wird.
