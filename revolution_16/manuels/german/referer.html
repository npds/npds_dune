HTTP Referer

Diese Funktion zeigt Ihnen woher die Besucher Ihrer Webseite kommen.

Sie können die Referer per Filter gruppieren : wenn filter=50 dann wird nach den ersten 50 Buchstaben gruppiert.

Denken Sie daran, die Liste hin und wieder zu löschen, damit sie nicht zu gross wird.

Sie können die Funktion in den Einstellungen deaktivieren.
Sie können auch die maximale Anzahl der Referer einstellen, damit Sie nicht irgendwann eine Fehlermeldung erhalten

Sie können die Liste archivieren, oder löschen (Pfad : slogs/referers.log).
