Persönliche Mitteilungen

Diese Funktion erlaubt die interne Kommunikation zwischen Benutzern. Das ist KEIN Newsletter! Wenn Sie wirklich die Newsletter Funktion benötigen, nutzten Sie Funktion Newsletter.
Sie können eine Nachricht an einen Benutzer, eine Gruppe, oder an alle Benutzer senden. Dazu geben Sie einen Betreff und einen Text ein.
