Testberichte


Dieses Modul erlaubt es Ihnen die "Testberichte" zu administrieren.

Sie können den Seitentitel einstellen, einen Testbericht und die Beschreibung (Mitteilung) einer Kategorie zuordnen. Klicken Sie auf Änderungen speichern.

Wartende Testberichte :
Zeigt die letzten Testberichte von Benutzern, Sie können Sie freigeben, oder nicht.

Einen neuen Testbericht hinzufügen :
Die selbe Seite, wenn ein Benutzer einen Testbericht hinzufügt.

Wenn Sie als Administrator eingeloggt sind, können Sie Testberichte direkt im Testberichtbereich editieren, oder löschen.
