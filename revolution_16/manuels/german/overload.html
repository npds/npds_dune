SuperCache

Dieses Modul erlaubt die Administration einer mächtigen Funktion in NPDS
SuperCache ist ein Php/Html Cache/Speicher System mit dem Sie die Geschwindigkeit Ihrer Seite optimieren können !
Sie können :

  • SuperCache Ein/Aus schalten,
  • Definieren Sie die Grösse, die Cachezeit (in Sekunden),
  • Leeren Sie den Cache vor einem grösseren Update Ihrer Seite.
