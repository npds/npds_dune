Administration – Logs

Auf der Registerkarte „Sitzungsliste“ können Sie die aktiven Sitzungen auf Ihrer Website im Detail anzeigen.
- Mit demSymbol können Sie die IP-Adresse trennen und sperren.
- Mit demermöglicht im Falle einer Verbindungsüberlastung beispielsweise das Trennen aller IPs.

Auf der Registerkarte „Protokollliste“ können Sie die vom Portal gesendeten Protokolle anzeigen und analysieren. Sie sind in zwei Kategorien unterteilt: sogenannte Sicherheitsprotokolle; und Protokolle zur Verfolgung von Downloads. In der Tabelle können Sie mit der Info-Funktion detaillierte Informationen zur IP erhalten.

Die Registerkarte enthält Befehle für:

  • Senden Sie die Datei security.log per E-Mail an den Superadmin.
  • Leeren Sie die Datei security.log
  • leere temporäre Upload-Dateien (modules/upload/tmp/)
