Abschnitte

Der Neue Abschnitte Manager hat eine grosse Funktionalität.

Die wichtigsten Funktionen sind:
 - eine bessere Baumstruktur: Abschnitt/Unter-Abschnitt/Publikation
 - minis-workflow der Publikation
 - eine gute Administration der Autorenrechte
 - eine gute Administration der Visualisierungsrechte

Bevor Sie beginnen:
 - zwei Positionen werden automatisch zum Zeitpunkt der Installation hinzugefügt:
   - Various erlaubt das erstellen einer Einzelseite (zum Beispiel)
   - Paper weight für die Dokumentenrechte, die Tests, die Einstellungen auf standby...
 - Ihre alten Abschnitte sind in Form von Sub-Kategorien verfügbar (ohne Titel) und die Autorenrechte sind einstellbar.
 - Es kann auch ein horizontales Javascript Menu undein spezifischer Block erzeugt werden(function#bloc_rubric)

::--::

Abschnitte:
 Es erzeugt einen Titel, eine Einführung und ein Statut: aktiv (ja oder nein).
Ein inaktiver Abscnitt wird nicht angezeigt in NPDS. Durch defekt erstellter nicht aktiver Abschnitt

Unter-Abschnitt:
Er ist eienm Abschnitt zugeordnet, zeigt einen Titel und (wenn nötig) ein Bild (Flash wird unterstützt :: speichern unter: images/sections), eine Einführung und Verwaltung der Benutzerrechte.

Publikation:
Ist einem Unter-Abschnitt zugeordnet und zeigt einen Titel It is attached to a sub-section and lays out a title, wenn nötig mit einem Link zu anderen Publikationen, Inhalte und die Benutzerverwaltung.
 NPDS Berechtigungen für den Inhalt: 
 - Code einfügen PHP, C, Perl.... Es ist nötig den Code zwischen Klammern zu setzten: Beispiel [code]your_source[/code ].
 - Die Administration mehrerer Seiten.  Es ist notwendig, die Seiten zu kennzeichnen via [ pageX ] wobei X folgendermassen aussehen soll [ page1 ] dann [ page2 ]...).

Neue Publikation:
  - Für den SuperAdmin:
   Die neue Publikation wird sofort veröffentlicht.
 - Für einen Administrator des Abschnitts:
   Die neue Publikation wartet auf die Veröffentlichung durch den Superadmin.

Pblikation im Zusammenhang mit:
Diese Funktion erlaubt eine Quer-Navigation zwischen verschiedenen Abschniten und Unter-Abschnitten.

Administration der Reihenfolge:
Da sind die Abschnitte, die Unterkategorien und die Publikationen. NPDS erlaubt Ihnen die Einordnung.

Status und Ausblenden:
Der Status eines Abschnitts wird angezeigt. Zur Erinnerung, ein Abschnitt mit dem Status 'Off' wird nichtt angezeigt in NPDS.
Das Ausblenden von Abschnitten, Unter-abschnitten und Publikationen wird nach einer Bestätigung.

Publkationen warten auf Veröffentlichung:
Die Liste zeigt die wartenden Publikationen und Sie sind in der lage diese zu: sehen, modifizieren, löschen oder freizugeben.

Autoren:
  - Für den SuperAdmin:
   Alles ist erlaubt

 - Für den Administrator des Abschnitts:
   Die Rechtevergabe muss, Unter-Kategorie für Unter-Kategorie vergeben werden(Vorsicht, wenn Sie eine neue Kategorie anlegen, die Unter-Kategorie : hat keine Rechte ist durch defect betroffen) beeinflusst durch 4 Kategorien:
   - Zu erstellen: Erlaube das Erstllen einer Publikation die dann auf Freigebe wartet
   - Zur Freigabe: Erlaube das Hinzufügen einer Publikation in die Warteschleife
   - Zu editieren: Erlaube das Editieren
   - Zu löschen: Erlaube das löschen

Jeder Autor kann seine Rechte an Kategorien und Unter-Kategorien einsehen.
 
Um die Baumstruktur zu aktivieren:
It Rechte zur Umsetzung der Anzeige von Unterkategorien und der Publikationen in der Baumstruktur..
