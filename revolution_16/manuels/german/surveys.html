Umfrage

Ändern Sie die Umfragen von Zeit zu Zeit. Denken Sie an Ihre Seitenbesucher.
Um eine Umfrage zu erstellen, füllen Sie das Formular des Umfrage Modul aus.

Sie können mehrere Umfragen auf Ihrer Seite anzeigen lassen (several blocks / pollnewest params) UND Umfragen in statischen Seiten (beispielsweise) mit Meta-Wort !block!... und metalang=1 als Option für static.php.

Titel

Die Frage zu Ihrer Umfrage

Optionen

Eine Umfrage erfordert minimal zwei Optionen (ja /nein) und Sie können (müssen aber nicht) alle Felder nutzen.

Eine Umfrage löschen

Klicken Sie auf Umfrage löschen. Dann suchen Sie die Umfrage aus, welche Sie löschen wollen. Die Umfrage wird sofort aus der Datenbank entfernt. Seien Sie also vorsichtig.

Eine Umfrage editieren

Sie können eine Umfrage editieren ohne die Abstimmung zu verändern. Suchen Sie eine Umfrage aus, die Sie editieren wollen.
Sie können die Umfrage so einstellen, das nur registrierte Benutzer abstimmen können und/oder die Umfrage schliessen. Die Datenbank wird sofort upgedated.
