Module Verwalten.

Ermöglicht die Installation und Deinstallation von Modulen. Die Schnittstelle präsentiert eine einfache Tabelle mit dem Symbol, dem Modulnamen und den Funktionen.

Funktionen

Für Module mit automatischer Installation
  • : Modul installieren.
  • : Deinstallieren Sie das Modul.
    Bitte beachten Sie, dass dieser Befehl zerstört, was durch die automatische Installation des Moduls erstellt wurde:
    • alle Tabellen in der Datenbank sowie die dort gefundenen Daten.
    • alle administrativen Funktionen und damit verbundenen Rechte
    • alle Blöcke, die sich auf das Modul beziehen.
    • beliebige Metawörter
  • Diese beiden Befehle sind alternativ nur verfügbar, wenn für das Modul eine automatische Installation in der Datei /modules/mon_module/install.conf.php definiert ist. Sie führen daher zur detaillierten und fortschrittlichen Schnittstelle zum Installieren oder Deinstallieren Ihres Moduls.
Für Module ohne automatische Installation
  • : Markieren Sie das Modul als installiert.
  • : Markieren Sie das Modul als deinstalliert.
  • Diese beiden Befehle sind alternativ nur verfügbar, wenn das Modul nicht über eine automatische Installation verfügt. Sie dienen daher lediglich dazu, den Status des installierten/deinstallierten Moduls in der Datenbank und dessen Benachrichtigung in der Schnittstelle zu ändern. Sie müssen das Modul daher installieren/deinstallieren, indem Sie auf die Datei install.txt aus dem Modularchiv verweisen.
