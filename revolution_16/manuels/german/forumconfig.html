Erlaubt, oder verbietet die Verwendung von Signaturen.

Erlaube die Anzeige heisser Themen mit verschiedenen GIF Grafiken.

Anzahl der Mitteilungen pro Seite :

Das ist die Anzahl der Postings/Themen pro Seite.

Themen pro Forum :

Anzahl der Themen in einem Forum (für ein Forum)

Aktivieren Sie die Sortierung der „gelösten“ Beiträge

Aktiviert die Sortierung und Anzeige des Links [(Nur / Ohne) Gelöst].

Maximale Anzahl von Beiträgen pro IP pro 30-Minuten-Zeitraum

(0=System inaktiv)
Mit dieser Option können Sie die maximale Anzahl von Beiträgen angeben, die dieselbe IP-Adresse ODER dasselbe Mitglied posten kann während:

  • eine anfängliche Dauer von 30 Minuten
  • ein zweiter Zeitraum von 90 Minuten (in diesem Fall wird der doppelte Wert verwendet)
  • 0 oder leer kehrt zurück zu; Deaktivieren Sie das System

Aktivieren Sie das Hochladen in Foren

Ermöglicht Ihnen, die Möglichkeit zu aktivieren um Dateien direkt in den Foren zu speichern. Dann können Sie diese Funktion nur in bestimmten Ihrer Foren autorisieren (siehe Verwaltung / Bearbeitung von Foren).

Versteckte Texte in Foren aktivieren

Ermöglicht Ihnen, die Möglichkeit zu aktivieren um einen Teil des Texts eines Beitrags in den Foren auszublenden, indem Sie die Tags [hide] Versteckte Texte [/hide] verwenden.

Text für Rolle X

Das Forum ist in der Lage, abhängig von der Rolle eines Mitglieds spezifischen Text anzuzeigen. In diesen Bereichen werden mehrsprachige Tags unterstützt.
