Administrationsseite

Einführung

NPDS ist ein französischer Open-Source-Portalgenerator unter der kostenlosen GNU-GPL-Lizenz mit PHP- und MySql-Technologien, für den in den Einstellungen ein mehrsprachiges Gerät konfiguriert werden kann. Standardmäßig ist das Portal in Französisch.

Der Inhalt und die Benutzeroberfläche dieser Seite hängen offensichtlich von den Rechten ab, die Sie als Administrator haben (der Superadministrator hat daher Zugriff auf alle Schnittstellenfunktionen).
Die allgemeine Schnittstelle ist in 3 Hauptblöcke unterteilt.

Warnungen und Benachrichtigungen

Der erste Block sammelt die Warnungen / Benachrichtigungen, mit denen Sie ausstehende Administratoraufgaben bearbeiten können. (vorgeschlagene Artikel, geplante Artikel, Benutzer, die auf Validierung warten, neue Links, defekte Links usw.)

Die Funktionen

Um NPDS zu verwalten, haben Sie Zugriff auf viele Schaltflächen, die nach einer Klassifizierung nach Typ nach Zeilen gruppiert sind. Sie haben die Form von Symbolen oder Text, abhängig von der Konfiguration der in den Einstellungen definierten grafischen Oberfläche.

  1. Mit den Schaltflächen in der ersten Zeile können Sie den Inhalt Ihrer Website verwalten. Sobald Sie auf eine Schaltfläche geklickt haben, müssen Sie auf die manuelle Online-Schaltfläche klicken, um die Dokumentation zur Öffnungsfunktion zu erhalten.
  2. In der zweiten Zeile können Sie Portalbenutzer verwalten
  3. In der dritten Zeile können Sie alles verwalten, was mit Kommunikation zu tun hat
  4. In der vierten Zeile können Sie die Schnittstelle der Blöcke verwalten
  5. In der fünften Zeile können Sie auf dem System agieren. Der Link "Einstellungen" ist einer der wichtigsten Links. Beachten Sie den Link, über den Sie ein Modul installieren und herausfinden können, welche Module bereits installiert sind oder nicht.
  6. Schließlich die letzte Zeile, in der Sie die mit einer Verwaltung ausgestatteten Module verwalten können.

Die letzten x Artikel

  • Zeigt die Liste der letzten x Artikel an, die auf Ihrer Site veröffentlicht wurden
  • Die erste Aufzeichnung ist der zuletzt veröffentlichte Artikel
  • Bestimmte Artikel können im Archiv angegeben werden. Dies sind die Artikel, deren Veröffentlichungsdatum abgelaufen ist.
  • Um diese Artikel wieder in den Feed aufzunehmen, müssen Sie sie nur bearbeiten, auf die Option "Datum ändern" klicken und ihr Enddatum ändern.
  • Klicken Sie auf den Artikel, um ihn zu bearbeiten, oder verwenden Sie die Links Bearbeiten, Löschen.
  • Wenn Sie ein altes Element bearbeiten / löschen möchten, geben Sie seine ID ein und wählen Sie die entsprechende Aktion aus.
