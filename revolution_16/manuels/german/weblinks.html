Weblinks


Dies ist eine bedeutende Anwendung in NPDS.
Damit können Sie eine Yahoo ähnlichen Linkindex erstellen, mit einer eigenen integrierten Suche.

Wartende Links :
Wenn ein Link auf die Freigebe wartet....
Sie können den Title, die URL, den Kommentar, den Namen und die E-mail Adresse des Linkeinsenders, sowie die Kategorie einstellen.
Dann können Sie den Link hinzufügen (Add) oder nicht (Delete).

Link DB reinigen:
Stellt die Bewertungen der Links auf null zurück.

Tote Links meldem :
Das sind die Optionen :
- Ignore löscht die Meldungen zu toten Links.
- Delete entfernt tote Links und die dazu abgegebenen Bewertungen.

Linkangeben ändern :
Für jeden Link haben Sie die original URL (die in der Datenbank gespeichert ist) und die neue URL.
Optionen sind :
- Ignore löscht alle Anfragen zum vorhandenen Link.
- Delete löscht tote Links und fügt die neuen URLs hinzu.

Eine Hauptkategorie hinzufügen :
Um die Links zu strukturieren, geben Sie einen Titel und eine Beschreibung ein (alphabetisch geordnet).

Unterkategorie hinzufügen :
Um die Links zu strukturieren, geben Sie eine Unterkategorie ein und ordnen Sie sie einer Hauptkategorie zu (eine Ebene / alphabetisch sortiert).

Neuen Link hinzufügen :
GEben Sie den Titel ein (Sie können HTML Code verwenden (br, img src (local or no), ...) wenn Sie wollen), URL, Kategorie (und Unterkategorie), Beschreibung, Name und E-mail des Melders. Klicken Sie auf Add.

Kategorie ändern :
Sie können den Namen einer Kategorie (Unterkategorie) ändern.
Sie können auch eine Kategorie (unterkategorie) löschen inclusive der enthaltenen Elemente,

Links ändern :
NPDS zeigt alle in der Datenbank vorhandenen Links an. Klicken Sie auf einen Link, um ihn zu ändern.
Sie können alle Parameter ändern, oder den Link löschen.
