Mitglieder editieren

Diese Anwendung erlaubt es Benutzer (Mitglieder) anzulegen, zu editieren, oder auch zu löschen.

Neuen Benutzer hinzufügen

Füllen Sie das Formular aus. Pflichtfelder sind mit einem * gekennzeichnet (Nickname, Email, Passwort).
Andere Felder sind keine Pflichtfelder. Die verborgene E-mail Adresse ist die, welche für andere Benutzer sichtbar ist(Profile, Foren, Beiträge).
GEBEN SIE NIE IHRE RICHTIGE E-MAIL ADRESSE in dieses Feld ein.
Klicken sie auf Benutzer hinzufügen und NPDS wird eine E-mail und ein Passwort an den neuen Benutzer senden.

Benutzer Rolle

Sie können dem Benutzer eine bestimmte Rolle zuweisen (zum Beispiel : Administrator, hot-line ...). Die Rolle des Benutzers wird im Forum und in der role'list is in forum' Konfiguration angezeigt.

Gruppe

Ein Benutzer kann Mitglied von 0 bis n Gruppen sein und der Gruppenwert ist zwischen 2(min) und 126(max). Gruppen werden durch Komma in der Datenbank getrennt.
- Beispiel : 2,25 (Mitglieder der Gruppe 2 und 25)
Die Liste der verfügbaren Gruppen wird angezeigt und erlaubt eine Mehrfachauswahl(shift / Ctrl).
Die erste Auswahl erlaubt es die Mitgliedschaft in einer Gruppe zu löschen.
Eine Gruppe ist ein Ebenenlink der seine Bestimmung beim anlegen von Blöcken findet (siehe Blöcke).

Verbindung erlauben

Normalerweise kann sich ein Benutzer nach Erhalt der E-mail mit seinem Passwort anmelden (Grundeinstellung ist YES (JA).
Ändern Sie die Einstellung auf OFF (AUS) um dem Benutzer den Zugang zu Bereichen die nur für Miglieder sind zu verbieten.

Benutzer editieren

Geben Sie seinen Nicknamen ein, wählen Sie Benutzer editieren und klicken Sie Ok.
Sie können alle Angaben verändern (einschliesslich des Passworts).

Benutzer löschen

Geben Sie den Nicknamen, wählen Sie Benutzer löschen und klicken Sie Ok.

Benutzer abmelden

Geben Sie den Nicknamen ein, wählen Sie Benutzer abmelden und klicken Sie Ok.
