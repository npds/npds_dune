Administration - META_LANG

META_LANG ? Wozu nutzt es?

Der META_LANG ist nutzbar für die Sprachanzeige.
Das erlaubt die Anwendung von META_LANG (in News, einem Artikel, einer Notiz), einem Kommentar oder im Edito oder gar in einer statischen Seite. Das ist das Resultat der Entwicklung von NPDS Language.

In Wissen es gibt 5 Typen um META_LANG für Ihre Seite zu nutzen

Die Typen von META_LANG

  • docu
    Wird verwendet um ein Wort von META_LANG auszuschliessen.
    Derzeit ist in dieser Kategorie nur ^ vorhanden
    Sie können diese Art von Metawort nicht erstellen
  • meta
    Wird in fast allen NPDS verwendet und dient dazu, das Ergebnis einer bestimmten Funktion aufzurufen, das an der Stelle seines Eintrags angezeigt wird.
    Dies ist die wichtigste Kategorie von NPDS, sie ermöglicht die Implementierung sehr erweiterter Funktionen.
    Abhängig vom Grad der Komplexität dessen, was Sie erreichen möchten, gibt es zwei Formen des Schreibens (im Skriptbereich):
    • $cmd=... PHP-Anweisung in einer einzigen Zeile ...;
    • function MM_dein-meta-mot () {
      code php
      code php
      code php
      return (ihre-return);
      }
  • mot
    Wird genutzt, um einem Wort (Buchstaben und Zahlen) eine Funktion zuzuordnen. Beispielsweise der Aufruf eines Links.
  • smil
    Wird verwendet, um Smilies und/oder Verknüpfungen zu Bildern zu integrieren.
  • them
    Wird verwendet, um interne NPDS-Funktionen im Zusammenhang mit Themen zu dokumentieren.
