编辑/创建一个功能框


NPDS，允许在页面上显示一个或多个功能框。
NPDS，允许在几乎所有页面上显示功能框（对应的主题下）。

一个功能框可以保留显示，仅于管理员，或仅注册会员，或给N个组的会员，也可仅保留于匿名者，当然也可以公开显示于所有人。 

 修改一个功能框：
 在功能框中修改必要的信息，并选择期望的操作，然后点击 ok 按钮。
 一次只能对一个功能框进行修改。 

 删除一个功能框：
 选中该功能框然后选择 删除一个功能框，最后点击 ok 按钮完成操作。一次只能删除一个功能框。 

创建一个功能框 (置左或置右)：
 输入一个功能框的标题，索引（index, 允许按顺序显示），及框中的内容，最后点击 创建一个功能框 按钮。
 系统默认为新建的功能框是被激活并有效的。

缓存功能框 :
   - 功能框涉及“超级缓存”，可以在不影响相关主题的文件的情况下被激活或关闭。
       缓存时间设置为0的功能框不被隐藏。
    - 在线帮助，可以去往“背景下帮助”功能，或使用PHP开发。 （如果您不确定，请保留该区域为空。） 

 NPDS的强大的 功能框系统 可以显示或执行很多功能：
 - HTML (要显示支持HTML的功能框，请在创建时选择HTML选项。)
 - Javascript
 - PHP 和/或SSI编码 (包括服务器端)
 - PHP 和/或NPDS的功能 

 只有在遵守以下文法的情况下，才能包含有编码：
 . 关于PHP 和/或SSI编码:         include#您的目录名/您的文件.php (无以下符号 ' 或 " )
         该文件必须是php，而返程必须写在 $content。 

 . 关于PHP功能 和/或NPDS的功能：
         function#您设置的功能 => function#Xblock (无以下符号 ' 或 " 或 () 或 ; ) 

         如您设置的功能有以下参数：
           function#您设置的功能 => function#XNews (无以下符号 ' 或 " 或 () 或 ; )
          params#您设置的参数 => params#5,10,false (无以下符号 ' 或 " ，但是以逗号分隔 ) 

           例如：
           function#category

            function#oldNews
           params#$storynum

            function#RecentForumPosts
           params#Forums Infos,15,0,false,10,false,-:    

 您也可以采取2步操作 :
         include#您的目录名/您的文件.php (无以下符号 ' 或 " )
         然后
         function#您设置的功能 => function#XNews (无以下符号 ' 或 " 或 () 或 ; )
         params#您设置的参数 => params#5,10,false (无以下符号 ' 或 " ，但是以逗号分隔 )
         => 这样，您可以输入编码并执行之！ 

 CSS 样式表单的功能框:
   某个式样设置的 class-title#名称 在您的CSS文件中.
   某个式样设置的 class-content#名称 在您的CSS文件中.
    => 以上2行命令允许为每个功能框定义一个CSS样式 (标题和内容)！
       - 动态主题有2个新的标记代码(meta-mots) : B_class_title 和 B_class_content。后2者可以应用在bloc.html的代码中。
        - PHP主题则有2个新的变量 : $B_class_title 和 $B_class_content。后2者可以被“引进使用”在 themesidebox 功能中 (通过变量 ...)  

 URI功能框:
   uri#uris_以_空格_分隔 用来显示注明的URI的功能框。
   在这种情况下，所维持时间的定义必须为0。
    例如 : uri#index.php user.php forum.php static.php?op=statik.txt&npds=1
  该 uri# 标记允许使用* （即*可代替所有文字）: 例如： viewforum.php?forum=* 或 viewtopic.php?topic=*&forum=10
   => 以上为系统唯一认可的文法！

Hidden Block :
 hidden#... to not show the theme of the blocks but load the content.
 This allows to use the block system for 'load' of javascript, PHP files, CSS files, ...
 Example : hidden#<script type="text/javascript" src="lib/yui/build/yui/yui-min.js"></script>
 => This syntax is the only one which is supported !
