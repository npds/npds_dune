论坛的参数设置


允许使用HTML :
 允许或禁止（删除）使用HTML代码。  

 允许使用 Smilies :
 允许在论坛中使用表情符号。  

 允许使用 les Signatures :
 允许注册会员在论坛的文章发表中使用他们自己的签名格式。  

 “热门话题“的访问量最低开始于 :
 此功能用于定义：从多少访问量开始，一个主题可以成为“热门话题”。  

 每页显示的帖子数量 :
 此功能用于定义：关于某个主题的帖子在同一页上显示的数量。  

 论坛中的主题数 :
 此功能用于定义：一个论坛可以有多少个主题。  

 激活根据`完成讨论`的状态选择帖子的功能 :
 激活根据帖子当前讨论结果的状态将其选择并显示的功能( 仅显示已结束讨论的帖子/显示所有帖子)。  

 同一个IP地址在30分钟内可以发表帖子的最多次数 (0=系统未激活) :
 此功能用于定义：同一个会员或IP地址最多能发表的帖子数量，在以下条件中 :
   - 在30分钟以内
   - 在第二个90分钟的阶段 (在这种情况下，本值的双倍将被采用)
 数字0 或 置空 将停止并关闭系统。  

 激活在论坛中上传 :
 此功能用于直接向论坛发送并存储文件(file)。 激活本功能后，您才可以在某些论坛中使用上传的功能 (见说明中的 管理/编辑论坛 )。  

 激活在论坛中隐藏文本 :
 此功能旨在激活隐藏文本的可能性，即：允许论坛中某个帖子的某个部分可以被设定为隐藏，只需使用编码如下：
[hide] 我所要求隐藏的文字部分 [/hide].

为某X设定的文本：
论坛根据某注册会员的身份而张贴一个专门的文本。 在本区域中，可以使用多语言标签语言 (Tag)。
