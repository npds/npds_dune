LNL - 短信息


 本模块允许您管理您的网站上的短信息LNL。  

 其中的主要功能如下：
   - 对短信的3个主体的管理：信头，信文本，信尾，
   - 可以对短信息进行文本格式或HTML格式的编制，
   - 发送给注册会员，潜在目标客户，或者两者皆可，
   - 短信息发送后的跟踪报告，和以往发送对象的记录列表。
 
 本功能采用典型的短信工作模式：
   - 创建信头 (文本格式 或 Html),
   - 编写信主体文本 (文本格式 或Html),
   - 创建信尾 (文本格式 或Html),
   - 测试该短信息 (输入相应的数字),
    本测试使您的短信可以被对方回复,
   - 发送该LNL短信息 (输入目标对象)。
 

 您可以（根据实际需要）添加，修改或者删除您的短信中的各个“组成部分”。 

 您可以观测到所发送的短信息的一些特征，如下：
   - All => 短信发送给注册会员和潜在目标客户,
  - Mbr => 仅发送给会员,
  - Out => 仅发送给潜在目标客户,
 同样的，您可以持有一个目标客户的列表 (OK => 用户期望收到信息；NOK =>用户不希望接收信息)。
 
 请注意，要使用本短信息功能，您的服务器上的邮件(Mail 或 Email)功能必须是激活的。 

 在拥有50个注册会员上时，LNL短信息系统将每10秒钟发送信息给50个会员。 此举目的在于避免服务器工作超负载。
