主功能框


所有的站点都有一个主菜单！您不能删除此功能框，且您仅有一个主菜单。

该功能框是您站点上所有功能的出发基础。
您可以在该主功能框中使用HTML代码，但是不能使用PHP代码。

介绍几个主要功能：

  • • a href=static.php?op=premier.txt => 统计页面
  • • a href=sdv.php?sdv=Annonces => 广告栏 (?sdv=[空间的名字])
  • • a href=sdv.php?sdv=Nom%20de%20Salle => 广告栏 (%20 用以代替空格)
  • • a href=index.php?op=index.php => 新闻 (op=[page de démarrage])
  • • a href=forum.php => 论坛
  • • a href=topics.php => 主题
  • • a href=sections.php => 栏目
  • • a href=reviews.php => 评论
  • • a href=links.php => 友好链接
  • • a href=download.php => 下载
  • • a href=memberslist.php => 通讯簿
  • • a href=faq.php => FAQ
  • • a href=friend.php => 向朋友推荐
