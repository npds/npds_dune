行政管理

简介

NPDS 是一个开源法语门户生成器，采用免费的 GNU-GPL 许可证，使用 PHP 和 MySql 技术，具有可以在首选项中配置的多语言功能。 默认情况下，门户使用法语。

该页面的内容和界面显然取决于您作为管理员所拥有的权限（因此超级管理员将有权访问所有界面功能）。 总体界面分为3个主要块

警报和通知

第一个块汇集了警报/通知，允许您处理待处理的管理员任务。 （提议的文章、预定的文章、等待验证的用户、新链接、损坏的链接等）

函数

要管理 NPDS，您可以访问许多按钮，这些按钮根据类型分类按行分组。 它们以图标或文本的形式出现，具体取决于首选项中定义的图形界面设置。

  • 第一行的按钮允许您管理网站的内容，单击按钮后，您必须单击在线手册按钮才能获取与打开相关的文档功能。
  • 第二行允许您管理门户的用户
  • 第三行可让您管理与通信相关的所有内容
  • 第四行允许您管理块的界面
  • 第五行允许您对系统进行操作，首选项链接是最重要的链接之一。 请注意允许您安装模块并了解哪些模块已安装或未安装的链接。
  • 最后一行允许您通过管理来管理模块。

最后 x 篇文章

  • 显示您网站上最近发布的 x 篇文章的列表
  • 第一条记录是最后发表的文章
  • 某些文章可以在存档中显示，这些文章已超过发布结束日期。
  • 要将这些文章放回 Feed，只需对其进行编辑，点击“更改日期”选项并修改其发布结束日期即可。
  • 点击文章进行编辑或使用编辑、删除链接。
  • 如果您想编辑/删除旧文章，请输入其 ID 并选择适当的操作。
