编辑论坛


本功能旨在便于您管理您的论坛。
 首先出现的列表显示的是您的网站的当前有效的类别和论坛。  

 编辑论坛：
 之后出现的第二个页面是在某个类别下的所有论坛。
 您可以添加一个论坛：

 - 索引 (Index): 和功能框一样，本功能是用以分类您的不同论坛。
 - 论坛名字
- 对论坛的描述 (请勿太冗长)
 - 版主的标识 (必填项): 如果一个论坛同时有多个版主，请以逗号划分他们的名字（,）。 

 - 广告的限制级别 :
 => 匿名广告被允许
 => 仅注册用户可以使用的广告
 => 仅版主可以使用的广告
 => 关闭 (论坛再不使用任何新的广告刊登或修改) 

 - 论坛类别 : 公众，私人，PHP Script, 文字扩展, 组 或隐藏
 => 公众= 对所有人公开
 => 私人= 使用密码进入
 => PHP Script= 可以存储和张贴编码(php, perl, c, 等等)的论坛。
  请注意某些功能在论坛中无效（微笑符号等）
 => 文字扩展= 允许在论坛中使用SFORM表单
 => 组= 允许在论坛中使用N个组，各组有自己的会员
=> 隐藏= 只有会员可以浏览到
 => 某些填写区域随条件不同而有区分 :
         -> 密码 (在“私人”的论坛中)
         -> 表单文件的文件名 (在“文字扩展”的论坛中)
                  在 modules/sform/forum (例如 : formulaire.php)
         -> 会员组 (在分组的论坛中) 的区分以逗号 (,) 为标记。// 例如 : 5,12,56  

 - 论坛形式 : 标准型，或树型
 => 树型结构允许较为特别的管理模式（树木状增长相关帖子），但是对系统的计算能力要求很高。 

 - 附件 : 如果在以下目录：管理/论坛设置(Administration / Configuration des Forums)中, 选项“激活在论坛中上传”为有效状态，
 => 则您可以允许该论坛有 粘贴附件 的特殊功能。 

 如要实现对某个外界论坛（某种连接）转移内容，只需在论坛名字的位置输入URL地址！  

 您可以编辑某个论坛。修改其中内容。
 您也可以删除某个论坛（系统将要求您再次确认）。  

 添加一个类别 :
 您可以添加一个类别，以便在该类别下建立论坛。然后，点击 添加 即可完成操作。  

 编辑某类别 :
 您可以由此修改某类别：其标题和ID标识（这实际就是分类时的参考标准）。点击 Ok 即可完成操作。  

 删除某类别 :
 本功能允许您删除某类别，及所有归属于该类别的论坛和帖子。
 系统会生成要求您再次确认的信息。
