Gestor de ficheros integrado



NPDS File Manager es un gestor de fichero simple (pero potente). Para poder utilizarlo, todos sus directorios deben estar en chmod 777. eso les permitirá evitar problemas para: escribir en los ficheros, crear nuevos ficheros y directorios.

Nota: No deje el FileManager activo (Admininistracion/Preferencias) cuando no lo necesite.

File Manager ya tiene su propia ayuda, esta zona de ayuda servira solo para saber rapidamente para que sirven algunos iconos.

Descripciones de las acciones
    Remontar en la estructura de los directorios.
    Borra el fichero o el directorio. Se pide una confirmación.
    Muestra el fichero en el navegador
    Editar el fichero (Plain text)
    Desplaza el fichero o el directorio hacia una localización que debe definirse.
    Crea un nuevo timestamp (fecha y hora corriente del servidor) para el fichero o el directorio.
