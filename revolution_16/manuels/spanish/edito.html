EDITO



Este módulo les permite simplemente publicar en vivo el EDITO de su pagina web.
Con el Editor HTML, puede hacer un copiar/pegar de Word (por ejemplo).

El nuevo módulo Edito de NPDS 4.8 P3 dispone de funcionalidades avanzadas:
- Edito Anónimo y/o Miembro,
- cantidad de días de visualización (a partir de la fecha y hora de modificación),
- Visualización de un texto para el día (08h00 = 20h00) o la noche
(20h01 = 07H59),


Por defecto:
- si no existe Edito Miembro, el Edito anónimo se indicará para los Anónimos Y los Miembros,
- el texto del Día solo se indicará el día si la casilla a [ día ] fue pinchada,
- el texto del Día se indicará el Día y la Noche si se hacen una cruz en las dos casillas y si no hay texto de Noche.,
- el texto de la Noche solo se indica la Noche,
- si los textos estan vacíos, el Edito no se indica.
