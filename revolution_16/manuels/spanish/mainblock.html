Bloque Principal

Todas las paginas Web tienen un bloque principal! No puede ni borrarlo ni tener mas de uno.

Este bloque es fundamental para poner los vinculos de sus paginas las mas destacadas.
Podra utilizar codigo HTML en este bloque, el codigo PHP esta prohibido y no podra utilizarlo.

Recordatorio de las principales funciones que podra poner en este bloque

  • a href=static.php?op=premier.txt => Pagagina estatica
  • a href=sdv.php?sdv=Annonces => Sala de ventas(?sdv=[nombre de la sala])
  • a href=sdv.php?sdv=nombre%20de%20sala => Sala de ventas (%20es el codigo para los espacios)
  • a href=index.php?op=index.php =>Las noticias (op=[pagina de inicio])
  • a href=forum.php => Foros
  • a href=topics.php => Asuntos
  • a href=sections.php => Rubriqas
  • a href=reviews.php => Las criticas
  • a href=links.php => Vinculos internet
  • a href=download.php => Descargas
  • a href=memberslist.php => Lista de miembros
  • a href=faq.php => FAQ
  • a href=friend.php => Recomiende esta pagina web
