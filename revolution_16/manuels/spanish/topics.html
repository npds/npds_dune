Gestion de asuntos y temas

Tiene que tener al menos un tema o un asunto para publicar correctamente sus noticias o articulos. Es un componente esencial del sistema. Este gestionario le permite añadir, editar o borrar un tema.
Cada tema debe de tener un titulo, una descripcion y una imagen (la imagen no es obligatoria) y el nombre de un administrador

Añadir un Tema

  • Titulo : Este titulo sera utilizado por NPDS. Inserte aqui un titulo en minusculas y sin espacios.
  • Descriptivo : Este es el texto que los internautas veran. Apliquese para que sea bien visto
  • Imagen : NPDS permite que no haya una imagen, pero si quiere poner una, inserte aqui el nombre de la imagen (ppor ejemplo: NPDS.gif).este fichero debe de estar en la carpeta images/topics en su cliente FTP.
  • Administrador : NPDS le permite poner uno o varios administradores.

Si deja esta zona vacia, SOLO los SuperAdministradores podran crear articulos y noticias en esta seccion.

Editar el Tema

Presione simplemente la imagen o el texto. En la página siguiente puede modificar el título, la descripción y la imagen.
Puede por eso suprimir el Tema pero recuerde que eso suprime también todos las noticias adjuntas (Mensaje de confirmación).

Sobre esta página puede también destinar vínculos, para un tema, a otros sitios Web (por ejemplo Php-Nuke). El vínculo relativo se indicará con los artículos relativos a este respecto.
