Administrar - Meta-lang.

META_LANG ¿para qué sirve?

META-LANG es un lenguaje específico de NPDS. Se compone de palabras (el META-MOT) que permiten, con total seguridad, llamar al lugar donde has introducido este META-MOT (en una noticia, un artículo, un post, un comentario o el editorial de tu sitio). o incluso una página estática), resultado de funciones internas de NPDS.

Hay 5 tipos de META-MOT en uso para su sitio

Tipos de META-MOT

  • docu
    Será utilizado por el CMS para procesamientos muy específicos y/o documentación en línea.
    Actualmente solo ^ está presente en esta categoría
    No podrás crear este tipo de metapalabra
  • meta
    En uso en casi todos los NPDS, se utilizará para llamar el resultado de una función específica que se mostrará en la ubicación de su entrada.
    Esta es la categoría más importante de NPDS y permite la implementación de funciones muy avanzadas.
    Existen dos formas de escritura (en el área de guión) dependiendo del nivel de complejidad de lo que se quiere lograr:
    • $cmd=... instrucción PHP de una sola línea...;
    • función MM_tu-meta-palabra () {
      código php
      código php
      código php
      devolución (tu-regreso);
      }
  • mot
    Se utilizará para mostrar una palabra (asociación de letras o números) que puede realizar una función básica de tipo HTML como llamar a un enlace, corrección ortográfica, gestión de abreviaturas. ...
    Esta categoría es muy útil para aumentar la productividad.
  • smil
    Se utilizará para integrar emoticones y/o accesos directos a imágenes.
  • them
    Se utilizarán para documentar las funciones internas del NPDS relacionadas con los temas.
