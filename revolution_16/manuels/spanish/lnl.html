LNL - Pequeña letra (Correo de informacion)

Este modulo le permite la gestion de la Pequeña letra LNL

Las funciones principales de esta funcion son:

  • Gestión de una Carta de información en 3 partes: el encabezamiento, el cuerpo, el pie.
  • Gestión de una Carta de información en en método texto o HTML.
  • Gestión de los envíos para los miembros, los clientes potenciales o los dos.
  • Seguimiento de las LNL enviadas y de la lista de los clientes potenciales.

Tipicamente el funcionamiento es :

  • Creacion del encabezado (Texto o Html),
  • Creacion del cuerpo(Texto o Html),
  • Creacion del pie de pagina(Creacion del cuerpo),
  • Test de la LNL(Introduzca el numero para cada uno de los elementos),
    Este test le permite verificar que su LNL esta perfectamente bien editada
  • Enviar LNL(Introduzca el Asunto para los interesados).
Puede añadir, modificar o borrar los"componentes" de su LNL.

Dispone de un estado de envios de LNL con varias informaciones notamente el Tipo :

  • All => LNL Enviado a todos los miembros y a todos los inscritos a la LNL,
  • Mbr => Solamente para los miembros,
  • Out => Solo para los inscritos la la LNL,
Asi que la lista de todos los inscritos (Estado OK => El usuario quiere recibir la LNL, NOK,No quiere recibirla).

Claro està, la funcionMail (o Email) debe de estar activadaen su dominio para poder utilizar esta funcionalidad.
Si tiene mas de 50 miembros, la LNL sera enviada por paquetes de 50 en 50 por periodos de 10 segundos cada uno, para no encombrar el servidor.
