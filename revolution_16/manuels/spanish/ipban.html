Administrar - IPban.

Este módulo principal le permite ver las IP prohibidas o en proceso de prohibición por npds antispam. También le permite prohibir manualmente IP (v4 o v6) o rangos de IP (v4).

Cada línea solo debe contener una dirección IP (v4 o v6) de la forma:
a.b.c.d|X (ej. v4: 168.192.1.1|5);
a:b:c:d:e : f:g:h|X (ej. v6: 2001:0db8:0000:85a3:0000:0000:ac1f:8001|5).
Si X >= 5, entonces esta dirección IP está prohibida y se denegará el acceso.

También puedes prohibir rangos de direcciones IP (v4) de esta forma:
x.x.%|5 o x.x.x.%|5
