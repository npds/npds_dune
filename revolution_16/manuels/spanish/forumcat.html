Edicion de los foros

Esta función le permitirá garantizar la gestión de sus Foros.
La primera lista indicada es en realidad la representación de las categorías y foros activos en su pagina web.

Edicion de los foros :
Una segunda página indica con la lista de los foros presentes en la categoría.
Pueden añadir un foro:
- Índex: de la misma manera que para los bloques, esto le da la posibilidad de clasificar sus distintos foros.
- Nombre del foro
- Descripcion del foro(No ponga nombres muy largos)
- identificante del moderador (es) (obligatorio): si hay varios moderadores para un mismo foro, separe los nombres por, (comas).

- Nivel de acceso para la publicacion :
=> Publicacion anonima autorizada
=> Publicación Usuario registrado solamente
=> Publicación Moderador solamente
=> Cerrado (se cierra el foro a toda nueva publicación o modificación)

- Tipo de foro : Público, Privado, PHP script, Texto extendido, Grupo u Ocultado
=> Publico = acceso a todo el mundo
=> Privado = acceso por contraseña
=> PHP Script=Foro que permite almacenar e indicar código (php, perl, c...).
Atención algunas cosas no funcionan con este foro (Caritas etc...)
=> Texto extendido= Permite asociar a un foro, un formulario SFORM
=> Grupo = Permite asociar un foro a N grupos de miembro
=> Texte escondido = visible que para los miembros
=> El Campo siguiente toma distintos valores:
    -> Contraseña (si Foro Privado)
    -> El nombre del fichero de formulario (si Foro Texto extendido)
       en modules/sform/forum (por ejemplo: formulaire.php)
    -> El identificador de los Grupos de miembro (si Foro Grupo) separar los nombres por, (comas)/ejemplo: 5,12,56

- Modo del foro : Standard o arbol
=> El método árbol permite una gestión particular (arborescente) de los posts pero puede requerir una determinada potencia de cálculo
- Documentos adjuntos : Si en Administración/Configuración de los Foros, la opción "Activar el upload en los foros" está sobre Sí
=> podrá especificar si autoriza, para este foro, los documentos adjuntos.

¡Para hacer una redirección hacia un foro externo (una clase de vínculo), inserte simplemente una url en el Nombre del Foro!

Sub-Forum :
A forum isn't visible if it's name start with <!>. One new meta-mot give you the reading status of one forum : forum_subfolder(ID_of_forum).
=> Imagine one forum with id = 3 and with this name : "<!>Only for NPDS" / Imagine the comment-text of forum with id 1 like : forum_subfolder(3) <a href="viewforum.php?forum=3">Only for NPDS</a>


Puede publicar un foro para modificarlo.
Puede suprimir un foro (mensaje de confirmación).

Añadir una categoría:
Le permite añadir una categoría en la cual podrá insertar foros. Presione Añadir para terminar.

Edicion (la categoría):
Les permite modificar la categoría: su título Y su ÍD (lo que permite clasificar las categorías). Presione Ok para terminar.

Borrar (la categoría):
Esta función le permite suprimir una categoría, todos los foros adjuntos así como todas las contribuciones.
Se indica a pesar de todo un mensaje de confirmación...
