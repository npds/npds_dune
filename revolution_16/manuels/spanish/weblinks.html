Base de datos de los Vinculos

La base de datos de los vinculos es una funcion importante de NPDS. su mantenimiento es simple pero necesitando de un cierto rigor
Esta opcion le permite crear una base de datos de vinculos internet, con un motor de busqueda como Yahoo por ejemplo.

Vinculos en espera de validacion

Dispondra de esta opcion solamente si un vinculo esta en espera (sic ...).
Puede modificar el titulo, la URL e incluso su comentario, el nombre y el correo de quien lo sugirio, la categoria.
Puede enfin decidir si publicarlo o no. (Boton Añadir) (Vinculo borrar ).

Limpiar la base de datos

Esta opcion le permite poner a cero la totalidad de los votos de los usuarios.

Señalizacion de vínculos rotos

Las ociones aplicables a la lista son :

  • Ignorar permite borrar todas las solicitudes para un vínculo dado.
  • Borrar permite borrar los vínculos rotos y los dictámenes para un vínculo dado.

Señalizacion de modificacion de vinculos

Para cada vínculo, tendrá el origen (actualmente almacenado en la base) y la propuesta de modificación. Las opciones aplicables son:

  • Ignorar permite borrar todas las solicitudes para un vínculo dado.
  • Borrar permite borrar los vínculos rotos y los dictámenes para un vínculo dado.
  • presionar el vínculo Autor o Proprietario para enviar un Correo electrónico.

Añadir una categoria principal

Para la estructuración del índice, señale un título y una descripción (clasificación por orden alfabético).

Añadir una subcategoría

Para la estructuración del índice, señale una subcategoría destinada a una categoría principal (un único nivel/clasificación por orden alfabético).

Añadir un nuevo vínculo

Coja el título (seguido de las balizas HTML (Br, img src (local o distante)...) si lo desean), la URL, la categoría (y subcategoría), la descripción, el nombre del proprietario y su correo electrónico. Presione el botón añadir.

Modificar la categoría

Esta función les permite modificar el nombre de una categoría y/o subcategoría.
Permite también suprimir una categoría y/o subcategoría y todos los elementos adjuntos (con mensaje de confirmación).

Modificar los vinculos

NPDS les presenta la lista de los vínculos actualmente disponibles en la base. Presione un vínculo para publicarlo.
Pueden modificar toda la información, incluido el número de hits (para colocar más ventajosamente un vínculo...). pueden también borrar el vínculo seleccionado.
