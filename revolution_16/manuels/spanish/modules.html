Administrar módulos.

Le permite instalar y desinstalar módulos. La interfaz presenta una tabla sencilla con el icono, el nombre del módulo y las funciones.

Funciones

Para módulos con instalación automática
  • : instalar el módulo.
  • : desinstalar el módulo.
    Atención, este comando destruye lo creado por la instalación automática del módulo:
    • cualquier tabla en la base de datos, así como los datos que se encuentran allí.
    • cualquier función administrativa y derechos asociados
    • cualquier bloque relacionado con el módulo.
    • cualquier metapalabra
  • Estos dos comandos están disponibles alternativamente sólo si el módulo tiene instalación automática definida en el archivo /modules/mon_module/install.conf.php. Por lo tanto, conducen a la interfaz detallada y progresiva para instalar o desinstalar su módulo.
Para módulos sin instalación automática
  • : marca el módulo como instalado.
  • : marca el módulo como desinstalado.
  • Estos dos comandos están disponibles alternativamente sólo si el módulo no tiene instalación automática. Por tanto, sólo se utilizan para modificar el estado del módulo instalado/desinstalado en la base de datos y su notificación en la interfaz. Por lo tanto, debe instalar/desinstalar el módulo consultando el archivo install.txt del archivo del módulo.
